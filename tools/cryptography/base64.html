Base64

Encode or decode Base64 strings or files

To encode or decode a string, paste it into the appropriate field and the other field will update automatically. Note that the encoded value is not encrypted, it is just a way to represent binary data in an ASCII string format. This means that anyone can decode it, so don't use it for sensitive data. If you need to encrypt data, use a proper encryption algorithm. If you need to encode or decode a file, use the Files tab.

To encode a file, drag it into the dropzone or click. The file will be encoded to base64 and you will be able to download it. To decode a file, paste the base64 string into the field below and click Decode and download file. Note that the encoded value is not encrypted, to encrypt data use a proper encryption algorithm.

Drag any file here or click to select and encode it to base64

You can convert one file at a time (up to 10mb in size)
